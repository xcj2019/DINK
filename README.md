DINK是新一代可实践应用落地的自动驾驶框架，包含传感器感知，定位，规划，决策，执行，各个模块通信及协同工作。适用于自动驾驶的各种场景及行驶速度。提供仿真环境，方便使用者模拟调试。

两大特点

1人工智能没有一点Caffe,纯Tensorflow

2全流程从传感器输入到CAN信号通信

两步安装

1安装Nvidia Docker

2运行DINK镜像

2 The whole process from sensor input to CAN signal communication
